NyarCode XLF Editor

Magnolia CMS Localization Tool

Extension for Visual Studio Code to easily view, filter, and edit XLF/XLIFF files from Magnolia CMS with a modern interface.

  • Two-column editing: original and translation
  • Instant filter with real-time search
  • Resizable and adaptable columns
  • Multiline textarea support
  • Dark/Light theme toggle
  • Modern and comfortable UI
